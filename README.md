# Pro Image Editor

一个功能强大的 Web 图像编辑器，提供接近 Photoshop 的专业图像编辑功能。

![版本](https://img.shields.io/badge/version-1.0.0-blue)
![许可证](https://img.shields.io/badge/license-MIT-green)

## ✨ 主要功能

### 🎨 绘图工具
- **画笔工具** - 自由绘制，支持调整大小和硬度
- **橡皮擦** - 擦除图像内容
- **油漆桶** - 智能填充相似颜色区域
- **吸管工具** - 拾取画面颜色

### 📐 选区工具
- **矩形选框** - 矩形区域选择
- **椭圆选框** - 椭圆/圆形区域选择
- **套索工具** - 自由绘制选区
- **选区操作** - 支持移动、删除选区内容

### 🖼️ 图层系统
- **多图层支持** - 无限图层数量
- **图层可见性** - 显示/隐藏图层
- **图层透明度** - 调整图层不透明度
- **图层混合** - 支持多种混合模式
- **图层操作** - 新建、复制、删除、合并图层

### 🎬 滤镜效果
- **模糊** - 高斯模糊效果
- **锐化** - 增强图像清晰度
- **边缘检测** - 提取图像轮廓
- **浮雕** - 3D浮雕效果
- **黑白** - 灰度转换
- **怀旧** - 复古色调
- **反色** - 颜色反转
- **噪点** - 添加噪点效果
- **马赛克** - 像素化处理

### 🎚️ 图像调整
- **亮度** - 调整明暗程度
- **对比度** - 调整画面对比
- **饱和度** - 调整色彩鲜艳度
- **色相** - 调整整体色调
- **色阶** - 精确色彩控制

### 🔄 变换操作
- **旋转** - 90°左转/右转
- **翻转** - 水平/垂直翻转
- **裁剪** - 自由裁剪画布
- **缩放** - 调整图像尺寸

### 📁 文件操作
- **新建** - 创建空白画布
- **打开** - 支持 PNG、JPG、GIF、WebP
- **保存** - 保存为项目文件
- **导出** - 导出为 PNG/JPG

### ⏪ 历史记录
- **撤销/重做** - 支持 50 步历史记录
- **历史面板** - 可视化历史操作列表

## 🚀 快速开始

### 在线使用
直接访问：[https://mlbclaw.github.io/pro-image-editor](https://mlbclaw.github.io/pro-image-editor)

### 本地运行
```bash
git clone https://github.com/MLBClaw/pro-image-editor.git
cd pro-image-editor
# 使用任意本地服务器打开，例如：
python -m http.server 8000
# 然后访问 http://localhost:8000
```

## 📖 使用指南

### 1. 创建新图像
点击菜单栏 **文件 → 新建**，设置画布尺寸后创建。

### 2. 打开图片
点击菜单栏 **文件 → 打开**，选择本地图片文件。

### 3. 使用画笔
1. 在左侧面板选择画笔工具
2. 在右侧面板调整画笔大小和硬度
3. 选择颜色
4. 在画布上绘制

### 4. 应用滤镜
1. 选择要处理的图层
2. 点击右侧面板的滤镜按钮，或从菜单栏 **滤镜** 中选择
3. 效果将立即应用到当前图层

### 5. 图层操作
- **新建图层**：点击图层面板的 "+ 新建" 按钮
- **调整透明度**：在图层面板修改透明度数值
- **合并图层**：选中图层后点击 "向下合并"

### 6. 撤销操作
按 **Ctrl+Z** 撤销上一步操作，或在历史记录面板点击任意步骤跳转。

## 🛠️ 技术栈

- **HTML5 Canvas** - 图像渲染核心
- **Vanilla JavaScript** - 无框架依赖
- **CSS3** - 现代化界面

- 🎨 **专业级界面** - 类 Photoshop 的操作界面
- ⚡ **高性能渲染** - 基于 Canvas 的硬件加速
- 📱 **响应式设计** - 适配不同屏幕尺寸
- 💾 **本地存储** - 所有处理在本地完成，保护隐私
- 🖥️ **跨平台** - 支持 Windows、macOS、Linux、移动端
